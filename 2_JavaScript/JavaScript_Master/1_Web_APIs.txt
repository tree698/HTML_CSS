#1. API : Application Programming Interfaces
  - 예) 자판기에 동전을 넣고 물품을 고르면 해당 물품이 나온다. 
          어떤 원리로 물품이 나오는지 몰라도 이용할 수 있다 

#2. 브라우저 제공 API
  - DOM APIs (Document Object Model: 웹페이지 요소 생성/삭제/스타일 변경)
  - Network APIs (서버와 통신)
  - Graphics APIs 
  - Audio/Video APIs (멀티미디어)
  - Device APIs (사용자가 온라인/오프라인인지 디바이스 상태정보 제공)
  - File APIs (사용자의 파일을 읽거나 저장)
  - Storage APIs (사용자의 정보 저장) 

#3. HTTP (Hyper Text Transfer Protocol)
#4. HTTPS (Hyper Text Transfer Protocol Secure)
